ESTE DOCUMENTO CONSTITUYE EL MANUAL TÉCNICO OFICIAL PARA LA IMPLANTACIÓN Y DESPLIEGUE DEL ECOSISTEMA POKEQUIZ.

1. FICHA TÉCNICA

  • PLATAFORMA: ANDROID NATIVO
  • SDK MÍNIMO: 29 (ANDROID 10.0)
  • SDK OBJETIVO: 36 (ANDROID 15+)
  • UI: JETPACK COMPOSE (RETRO GBA STYLE)

2. ARQUITECTURA

FRONTEND (ANDROID)

SISTEMA BASADO EN COMPONENTES RETRO PERSONALIZADOS. COMUNICACIÓN VÍA RETROFIT 2 Y PROCESAMIENTO DE IMÁGENES CON COIL.

BACKEND

NODE.JS + EXPRESS + PRISMA ORM. BASE DE DATOS POSTGRESQL EN RAILWAY.

3. REQUISITOS

SE REQUIERE ANDROID STUDIO (LADYBUG+) Y JAVA JDK 11 CONFIGURADO. ESPACIO MÍNIMO EN DISCO DE 4GB PARA HERRAMIENTAS DE COMPILACIÓN.

4. CONFIGURACIÓN

CLONACIÓN DEL REPOSITORIO

git clone https://github.com/diegofg11/Pokequiz.git

SDK ANDROID

CONFIGURAR EN local.properties:

sdk.dir=C\:\\Users\\USUARIO\\AppData\\Local\\Android\\Sdk

5. SERVICIOS

LA URL BASE DEL SERVIDOR DE PRODUCCIÓN SE GESTIONA EN EL ARCHIVO Network.kt. 
const val BASE_URL = "https://pokequizbackend-production.up.railway.app/"

6. TROUBLESHOOTING

ERROR SOLUCIÓN
CONEXIÓN REVISAR ESTADO EN RAILWAY O URL EN NETWORK.KT
SDK 36 INSTALAR DESDE EL SDK MANAGER DE ANDROID STUDIO
GRADLE SINCRONIZAR PROYECTO CON ARCHIVOS DE GRADLE
